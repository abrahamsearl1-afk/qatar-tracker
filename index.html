@Qatar Production Tracker

Connect to Firebase to enable live shared access for your team. Takes about 5 minutes — one-time setup.

1 Create a free Firebase project
Go to console.firebase.google.com and sign in with a Google account. Click Add project, name it qatar-tracker, skip Google Analytics, and click Create project.
2 Create a Realtime Database
In the left sidebar click Build → Realtime Database → Create database. Choose Start in test mode and click Enable. Leave the default region.
3 Get your config
Click the ⚙️ gear → Project settings. Scroll to "Your apps", click the </> Web icon, register an app name (e.g. tracker), and click Register app. Copy the firebaseConfig object — you'll paste it below.
